pub fn register_type_dyn( &mut self, type_id: TypeId, factory: Box<dyn FnOnce() -> Box<dyn DynamicInternTable>>, ) -> BaseValueId

Register a base value type given its TypeId and a factory closure that produces a fresh typed intern table. Used by the trait-crate-level BaseValuePool::register_type_dyn to route from a dyn-friendly API back to the typed BaseInternTable<P> constructor without exposing P: BaseValue to the caller.

If a type with the given TypeId is already registered, returns the existing BaseValueId and discards the factory.

pub fn intern_dyn_by_id( &self, id: BaseValueId, val: &(dyn Any + 'static), ) -> Value

Intern a base value through dynamic dispatch.

val must be a &P whose TypeId matches the type previously registered under id. The caller is responsible for honoring the P::MAY_UNBOX short-circuit; this method always goes through the intern table.

Used by egglog-backend-trait’s BaseValuePool::intern_dyn.

Source

pub fn unwrap_dyn_by_id( &self, id: BaseValueId, v: Value, ) -> Box<dyn Any + Sync + Send>

Unwrap a base value through dynamic dispatch.

Returns a Box<dyn Any + Send + Sync> holding a value of the registered Rust type for id. Used by egglog-backend-trait’s BaseValuePool::unwrap_dyn.

Source

pub fn intern_table_len_by_id(&self, id: BaseValueId) -> Option<usize>

Backing intern-table length for a NON-unboxable boxed type (its handle is a direct table index), or None for an unboxable type. See DynamicInternTable::intern_table_len.
