pub type PanicMsg = Box<dyn FnOnce() -> String + Send>;
Expand description

A lazily-computed failure message for RHS lookups / external calls.

The message is only built if the lookup or call actually fails at runtime. Deferring it matters because the usual message form is format!("{span}: ..."), and formatting a Span calls SrcFile::get_location, which scans the source-file prefix. Building it eagerly for every action in every rule is O(rules * file) — quadratic when many generated rules (term/proof encoding) are compiled against a large source file. Boxed so the trait stays dyn-compatible.
