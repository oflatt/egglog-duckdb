Configuration for the DuckDB backend.

The two flags here govern how EGraph::with_duckdb_backend sets up its underlying egglog_bridge_duckdb::EGraph and how the egglog frontend instruments the program before handing it off:

  • native_uf toggles the in-process union-find replacement for the term-encoded @_u_f___<sort>f table.
  • proofs runs term encoding in proof-tracking mode.
